JeffTorch

A lightweight, PyTorch‑shaped neural network library built for cross‑domain AI research.

JeffTorch is a minimal neural network framework designed to mirror the public API of PyTorch while remaining small, transparent, and easy to extend. It provides a clean, NumPy‑backed implementation of Tensors, Modules, Layers, Optimizers, and Functional losses. The goal is to offer a reusable, cross‑domain foundation for experiments, solvers, agents, and research systems.

JeffTorch is intentionally simple: no autograd (yet), no CUDA(yet), no heavy dependencies. Just the essential building blocks with a familiar interface.
